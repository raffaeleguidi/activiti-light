# activiti-light

requirements:

* mysql on port 3306
* an user called "root" with pwd "root"
* a database called "activiti"

to start the engine

* ./start.sh
* http://localhost:8080/activiti-rest

to start activiti-explorer

* cd activiti-explorer
* mvn tomcat7:run
* http://localhost:8081/activiti-explorer
